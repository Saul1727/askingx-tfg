El dashboard está fallando con el error "TypeError: asksList.filter is not a function". Esto ocurre porque el backend devuelve la respuesta envuelta en un objeto `{ success: true, data: [...] }`, pero el frontend está devolviendo el objeto completo en lugar de extraer el array. 

Para solucionarlo, por favor reemplaza COMPLETAMENTE la función `getAllAsks` por el siguiente código exacto (fíjate en el `return data.data || data;`):

export const getAllAsks = async (status = '') => {
  try {
    const url = status ? `${API_BASE_URL}/asks?status=${status}` : `${API_BASE_URL}/asks`;
    const response = await fetch(url, {
      method: 'GET',
      headers: {
        ...getAuthHeader()
      },
    });

    const data = await response.json();
    if (!response.ok) throw new Error(data.message || 'Error al recuperar las peticiones');
    
    return data.data || data;
  } catch (error) {
    console.error('GetAllAsks service error:', error);
    throw error;
  }
};
